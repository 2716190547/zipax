## zipax 0.22

zipax 0.22 is the public open source release polish pass.

### English

- Added MIT license metadata.
- Added bilingual support documentation and GitHub funding metadata.
- Improved custom Tauri window chrome on macOS.
- Increased the outer window radius and simplified window control buttons.
- Fixed bottom spacing by measuring the shell directly.
- Rebuilt the tray icon from an SVG source for sharper menu bar rendering.
- Kept active development focused on the Tauri cross-platform app.

### 中文

- 新增 MIT 开源许可证。
- 新增中英双语支持文档和 GitHub funding 配置。
- 改进 macOS 下的 Tauri 自定义窗口外壳。
- 调大窗口外层圆角，并简化关闭/最小化按钮样式。
- 通过直接测量窗口 shell 修复底部多余间距。
- 基于 SVG 源重新生成菜单栏托盘图标，显示更清晰。
- 后续维护聚焦 Tauri 跨端版本。
