# zipax v0.25.0

zipax v0.25.0 是特性更新，包含 Rust 压缩管道优化、文件夹监听改进和新增依赖页。

## 更新内容

### Rust 压缩管道优化
- 重构压缩 pipeline，减少不必要的格式探测和 I/O 操作。
- AVIF/HEIC/JPEG/PNG/WebP 编码路径优化。
- 新增 `ImageData` 统一中间表示，减少重复解码。

### 文件夹监听改进
- watcher 稳定性提升，减少重复触发和误报。
- 文件稳定性等待逻辑优化。

### 新增依赖页
- 桌面端新增 Dependencies 标签页，展示应用依赖信息。

### 官网更新
- 首页和组件样式优化，新增多个页面组件。
